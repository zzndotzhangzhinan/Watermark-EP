# Watermark-EP

This repository contains the Python code to reproduce numerical experiments in the paper: ``Online LLM Watermark Detection via E-processes''. Note that the code in this repository may not exactly replicate some of the results in the paper because of the randomness in simulating the data.

There are several folds in this repository:

1. Fold ``simu'' contains code for the simulation results in the paper.
2. Fold ``llm'' contains code for implementing the detection methods on open-source LLMs.
